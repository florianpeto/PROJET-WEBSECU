Bonjour  voici un resume de comment lancer mon projet

 1- Lancer l'importation de la base de donnee (registration_eleve)
 2- Ouvrir les fichiers create, add, config
 3- Lancer le serveur avec php -s localhost (le port depend de vous :!) )
 4- Entrer la page correspondante ( config.php)
 5- Entrer les valeurs selon le formulaire 
 6- Verifier l'enregistrement de la base de donnee 
 7- Boom c'est fini :)
